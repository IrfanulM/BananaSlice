# Contributing to BananaSlice

All contributions are welcome, major or minor! Whether you're reporting a bug, requesting a feature, or submitting a pull request, your help is appreciated.

## Setup

```bash
git clone https://github.com/IrfanulM/BananaSlice.git
cd BananaSlice
npm install
npm run tauri dev
```

**Prerequisites**: Node.js 18+, Rust

## Bug Reports

Open an issue with:

- Steps to reproduce
- Expected vs actual behavior
- Your OS and app version

## Feature Requests

Open an issue describing the feature.

## Pull Requests

1. Fork and create a branch
2. Make your changes following existing code style
3. Test with `npm run tauri dev`
4. Open a PR.
